# Analysis Figure Rationale

This note records why each model-dynamics figure was added and which project
references it is intended to support. It separates plotting choices from model
results so the dissertation can later cite the figures cleanly.

## Tuned Continuous Model Figures

These figures are generated by:


## Figure Purposes

| Figure | Why we added it | Reference anchor |
| --- | --- | --- |
| Fixed-point ring in PCA space | Shows whether task trajectories and fixed-point endpoints occupy a circular memory manifold. This is the main visual evidence that the tuned model has learned a ring-like state-space structure. | Kilpatrick, Ermentrout & Doiron 2013 for persistent bump activity and attractor stability; Ghazizadeh & Ching 2021 for trained RNN manifold/attractor analysis. |
| Decoded angle over time | Shows whether the remembered angle is decoded and maintained across cue, delay, and response. This connects hidden dynamics to the actual memory variable rather than only to PCA geometry. | Murray et al. 2017 for stable population coding; Ghazizadeh & Ching 2021 for delay-period decoding and forgetting dynamics. |
| Jacobian spectrum | Shows local recurrent stability around sampled fixed points. For a discrete-time ring-like attractor, the expected pattern is one near-neutral mode near eigenvalue `1`, with most other modes inside the unit circle. | Ghazizadeh & Ching 2021 for fixed-point/attractor-landscape analysis; the Yang/GY RNN dynamical-system notebook for the fixed-point plus Jacobian visualization workflow. |
| Perturbation recovery | Tests whether off-manifold hidden-state perturbations return toward the sampled ring under blank-delay dynamics. This is not stochastic noise analysis; it is deterministic basin/recovery analysis. | Kilpatrick, Ermentrout & Doiron 2013 for attractor robustness and drift; Ghazizadeh & Ching 2021 for robustness of working-memory manifolds. |
| Hidden-state movie | Shows the trial-time story in one animated view: the cue population drives the hidden state, delay dynamics settle near the sampled ring, and the response readout decodes the remembered angle. A separate adjacent panel shows all fixed-point-landscape perturbed late-delay starts moving under blank recurrent dynamics back toward the sampled ring. Smooth movie frames are interpolated between real recurrent model steps, so this is an explanatory visualization of the same sampled hidden trajectories, fixed-point ring, and perturbation-recovery logic, not a separate statistical test. | Same attractor/manifold anchors as the fixed-point ring and perturbation recovery, plus Murray et al. 2017 for relating evolving population states to stable decoded memory content. |

## Interpretation Boundary

These plots strengthen the attractor interpretation, but they still do not
exhaustively map the full `64`-dimensional hidden-state space. The correct claim
is that the stable tuned checkpoint shows sampled evidence for a ring-like
working-memory attractor: low-drift decoded angles, fixed points arranged on a
ring in task PCA space, near-neutral local Jacobian modes, and deterministic
recovery from moderate hidden-state perturbations.

Noise-driven drift/diffusion analysis is intentionally not included here. That
should be added separately when the project is ready to test stochastic
robustness and psilocybin-inspired variability.
